# custom-nba-team
A full-stack web application that allows users to create custom NBA teams and analyze their combined strength using real player statistics. Built with Python (Flask), NumPy, Pandas, and vanilla JavaScript.

Statistical Analysis: Calculate team strengths across multiple categories:

Shooting (Field Goal % + 3-Point %)
Scoring (Points per game)
Playmaking (Assists)
Defense (Steals + Blocks)
Rebounding
Overall Team Rating

How to Use:
1. Search for players by name (e.g., "LeBron James", "Stephen Curry")
2. Select 5 players to build your team
3. Click "Analyze Team Strength" to see comprehensive stats
